Installation
============

.. code-block:: bash
   :class: ignore

   pip3 install pip-versions


Usage
=====

Get the latest version

.. code-block:: bash
   :class: ignore

   pip-versions latest django


.. code-block:: bash
   :class: ignore

   3.1.2


Get all the versions

.. code-block:: bash
   :class: ignore

   3.1.2
   pip-versions list django
